Canvas System Icons

The following icons are pulled from the Canvas LMS. This HTML code resource is intended to allow users of the Canvas LMS to effectively identify and leverage Canvas icons within their eLearning course sites.

Be mindful that these icons do not have any contextual value. If these icons are being used independently have have interactive and/or contextual value, consider using role="img" aria-description="DESCRIBE THE ACTION WITH THE ICON". If these icons can be ignored by screen readers, include aria-hidden="true" to hide the element from screen readers.

Examples Using Icons

The <i> element is used to display icons from the Canvas Icon library by combining it with specific CSS classes that reference the desired icon. Although <i> originally represented "italic" text, it's commonly used for icons due to its semantic flexibility and lightweight HTML structure.

Icon in a Heading Example

Heading with an Icon Example
HTML: Result:
<h3>
  <i class="icon-flag icon-Solid" aria-hidden="true"></i> Examples Using Icons
</h3>

Examples Using Icons

Explanation:

  • <i> Tag: The <i> element holds the icon and provides a way to style it with CSS.
  • icon-* Class: This class call that initiates the icon library, allowing other icon-specific classes to work.
  • icon-flag Class: Specifies the exact icon (in this case, a flag). Each icon has a unique class, like icon-alerts, icon-quiz, etc.
  • icon-Solid Class: For each icon, there is a companion solid or filled icon. Calling icon-Solid Class instructs the browser to use the filled icon.
  • aria-hidden="true": Hides the icon from screen readers when it's purely decorative. This improves accessibility, as screen readers won't attempt to interpret non-essential icons.

Icon in a Button

Button with Icon Example
HTML: Result:
<a href="#" class="btn btn-primary">
  <i class="icon-like" aria-hidden="true"></i> Like
</a> 		Like

Explanation:

  • Button with Icon: By embedding the <i> element within a button, you add visual context, such as a thumbs-up, which is immediately recognizable as a "Like" action.
  • Positioning: Placing the <i> element before the text aligns the icon to the left of the button label.
  • Bootstrap Integration: The button uses the btn btn-primary classes to apply Bootstrap's primary button styling, with the Canvas icon enhancing its visual appeal and interactivity.

Practical Use

Using icons with <i> elements enhances user interfaces by adding easily recognizable symbols, improving clarity, and often making interfaces feel more modern and polished. The combination of Bootstrap and the Canvas icon set provides a cohesive and visually appealing user experience.

This approach can be applied broadly, from buttons and headers to navigation elements, as it keeps the HTML lightweight and ensures icons are easily customizable via CSS.
